# Video Game Sales Analysis

An Exploratory Data Analysis (EDA) project on 16,000+ video games.

## Goal
The goal of this analysis was to load a dataset, analyze it using Pandas, and find insights into sales trends by genre, platform, and title.

## Analysis
* Loaded and cleaned a 16,000-entry CSV file using **Pandas**.
* Performed data aggregation (groupby) to calculate the total sales for each game genre.
* Used **Matplotlib** to visualize the findings.
